# 🚀 To Do List & Productivity Tracker

A productivity-focused React web app that helps users stay organized and focused through:

* ✅ Priority-based task management
* 📅 Daily habit tracking
* ⏱️ Pomodoro study timer with break sessions

---

## ✨ Features

* Add, edit, and delete tasks
* Set different priority levels for tasks
* Mark tasks as completed
* Track daily habits and routines
* Pomodoro timer for focused study/work sessions
* Automatic break timer support
* Clean and responsive React UI

---

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* CSS

---

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/malak-darwish/ToDoList.git
```

Navigate into the project folder:

```bash
cd ToDoList
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 🌐 Open in Browser

The app will run at:

```bash
http://localhost:5173
```
